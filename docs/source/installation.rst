Installation
============

Clone the repository:

.. code-block:: bash

   git clone https://github.com/your-username/cine-to-nerdle.git
   cd cine-to-nerdle

Create a virtual environment:

.. code-block:: bash

   python -m venv .venv
   source .venv/bin/activate

Install dependencies:

.. code-block:: bash

   pip install -r requirements.txt
